# Implementation Plan

1. **Set up the project structure**
   - Initialize Python virtual environment
   - Create requirements.txt
2. **Choose GUI framework**
   - Use PySide6 for modern look and Markdown rendering
3. **Implement basic UI**
   - File open dialog
   - Area to display rendered Markdown
   - Simple menu (Open, Exit)
4. **Integrate Markdown rendering**
   - Use `markdown` library to convert .md to HTML
   - Display HTML in a web view widget
5. **Test with sample Markdown files**
6. **Iterate**
   - Add features (live reload, dark mode, etc.) as needed
