# Project Overview

MDViewer-ktb is a simple Markdown viewer for macOS, designed for rapid development and iterative improvement. The project leverages Python and PySide6 for a modern, cross-platform GUI and uses Python's markdown libraries for rendering.

## Goals

- Fast, minimal Markdown viewer for macOS
- Easy to extend and improve
- Clean, simple UI

## Key Features

- Open and view Markdown files
- Render Markdown as HTML in a web view
- Simple menu (Open, Exit)
- Iterative feature additions (e.g., live reload, dark mode)

## Development Environment

- Python (with virtual environment)
- PySide6 for GUI
- markdown for Markdown parsing
- Version control via GitHub
